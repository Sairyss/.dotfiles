# Kanata Typing Layer Cheatsheet

## Steno chords (always active)

| Chord         | Word     |
| ------------- | -------- |
| `u` + `i`     | `the`    |
| `u`+`i`+`o`   | `that`   |
| `i` + `o`     | `this`   |
| `u` + `o`     | `these`  |
| `m` + `,`     | `and`    |
| `f` + `o`     | `for`    |
| `a` + `r`     | `are`    |
| `y` + `u`     | `you`    |
| `w` + `a`     | `was`    |
| `w` + `i`     | `with`   |
| `w`+`i`+`l`   | `will`   |
| `m`+`,`+`.`   | `not`    |

---

## How it works

- **`jk`** chord → typing layer
- Exit anytime with **`esc`** or **`'`**
- **Commands**: double-tap the letter (`cc` = create, `ff` = find, ...)
- **Nouns**: two different letters (`fn` = function, `co` = code, ...)
- Any unbound key loops back to the typing layer start
